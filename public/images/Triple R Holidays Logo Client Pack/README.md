# Triple R Holidays Logo Client Pack

Official source files used: `Logo.svg` and `Logo.pdf`
Raster fallback/source dimensions: 2763 x 900px
Trimmed master dimensions: 2747 x 863px
Clear-space master dimensions: 3517 x 1105px

## What to send/use

- `01-master/` contains PNG masters, including a clear-space version.
- `02-web/` contains transparent PNG/WebP plus branded background JPG/WebP exports.
- `03-favicon-app-icons/` contains favicon/app icon PNGs and `favicon.ico`.
- `04-print/` contains 300 DPI print exports and the official PDF supplied by the client.
- `05-official-source-files/` contains the supplied SVG and PDF source files.
- `06-brand-background-variants/` contains white, Ivory Sand, and Ceylon Navy background versions with clear space.

## Note

The supplied SVG contains an embedded raster image rather than editable vector paths. The official SVG/PDF are included as supplied, and raster exports use the supplied SVG image data where available.
